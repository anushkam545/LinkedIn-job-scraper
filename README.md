# 🚀 LinkedIn Job Scraper  

## 📌 Project Overview
This project focuses on **web scraping job listings** from LinkedIn to analyze the current job market trends. It extracts key information such as job titles, companies, locations, and required skills, helping students and professionals understand **in-demand skills and hiring patterns**.

---

## 🎯 Objectives
- Scrape real-time job data from LinkedIn
- Analyze job roles and required skills
- Identify high-demand technologies
- Build a dataset for further data analysis and visualization

---
## 🛠️ Tech Stack
- **Python**
- **Requests** – for sending HTTP requests  
- **BeautifulSoup** – for parsing HTML  
- **Pandas** – for data cleaning and analysis  
- **Time** – to handle delays and avoid blocking  

---

## ⚙️ How It Works
1. Define job roles (e.g., Data Scientist, Data Analyst, etc.)
2. Send HTTP requests to LinkedIn job search pages
3. Parse HTML content using BeautifulSoup
4. Extract:
   - Job Title
   - Company Name
   - Location
   - Job Link
5. Store data in a structured format (CSV)
6. Perform basic analysis using Pandas

---

## 📊 Dataset Details
The generated dataset (`Output-dataset.csv`) contains:

- **Job Title**
- **Company Name**
- **Location**
- **Job URL**

This dataset can be used for:
- Skill demand analysis
- Salary insights (if extended)
- Location-based job trends
